# 1. 背景與核心問題

大型社交平台（Meta、YouTube 等）多年來允許大量詐騙廣告投放，造成全球性傷害。

這些並非普通詐騙，而是：

- AI Deepfake 名人假影片
- 偽冒金融投資平台
- 偽裝銀行／政府通知
- 偽裝主持人、YouTuber、金融專家
- 高回報投資詐騙（假保證回報）

核心問題不是詐騙者，而是：

> **平台對詐騙廣告審核不重視，且存在收入誘因，不願徹底處理。**

---

# 2. 為什麼詐騙廣告會大量出現？

### **A. 嚴格審核會減少廣告收入**
詐騙廣告願意出價極高的 CPC / CPM。  
平台越放任，收入越高。

### **B. 推高整體 CPC（全球合法廣告主全部受害）**
Scam Ads 願意付高價 → 推高所有平台廣告成本。

### **C. 審核成本極高（平台不願投入）**
影片審核需要：
- OCR
- 多幀分析
- AI Deepfake 檢測

平台為降低成本而選擇忽視。

### **D. 詐騙 ROI 極高**
- 成功詐騙一人：獲利 10 萬–100 萬美元  
- 廣告成本：甚至不到 100 美元

詐騙成為「全球最高 ROI 產業」。

# 9. 我們的訴求

1. **平台必須啟用 AI 審核系統**
   - Link 審核（包含跳轉後動態變更）  
   - 影片逐幀 OCR + AI 偵測可疑內容  
   - Deepfake 檢測模型

2. **用戶檢舉獎勵制度（扣除廣告主保證金）**
   - 若廣告被判定為詐騙  
   → 保證金扣除並獎勵檢舉者  
   → 大幅增加詐騙成本

3. **平台必須公布 Scam Ads 透明報告**

4. **制定國際統一法律框架（如 GDPR 等級）**
